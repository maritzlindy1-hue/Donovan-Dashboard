# Donovan JC Dashboard v2

## Render settings
- Build command: `pip install -r requirements.txt`
- Start command: `gunicorn app:app`

- requirements.txt
- render.yaml
- Donovan_JC_Live_Dashboard.xlsx
- templates/index.html
